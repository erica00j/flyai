## 프로젝트 소개

떠오르는 웹소설 시장을 겨냥해서 요즘 인기있는 숏폼 마케팅에 적용할 수 있는 숏폼을 제작해주는 서비스이다. 웹소설의 줄거리나 등장인물 등 정보를 입력하면 생성AI를 활용하여 숏폼 이미지를 제작한 후 BGM, 자막, TTS, Novel T만의 장면 효과를 삽입하여 약 1분 가량의 영상을 제작해준다.
